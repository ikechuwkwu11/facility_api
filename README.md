Facility & Booking API

This is a RESTful API built using **Django** and **Django REST Framework**. It allows users to register, log in, and manage **facilities** and **bookings**. The project is useful for systems that require booking and scheduling of facilities like hotels, conference rooms, or events.

Features
- User registration and login
- Facility management:
- Add, view, edit, and delete facilities
- Booking system:
- Add, view, edit, and delete bookings
- Logout functionality

Tech Stack
- Backend: Python, Django, Django REST Framework
- Authentication: Custom serializers for registration & login
- Database: SQLite (default), can be changed
- API Testing: Postman or any REST client
